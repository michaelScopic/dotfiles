# HTOP

This is my htop config. 

## Features:

- CPU temp
- CPU frequencies 
- Swap
- ZRAM
- Battery
- Disk I/O
- Network I/O
- Uptime

## Installation

(Inside the dotfiles directory...)

```sh
mkdir ~/.config/htop
cp config/htop/htoprc ~/.config/htop
```
